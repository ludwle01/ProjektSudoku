Purpose
-------

This toolbox was created to support a user with solving a sudoku puzzle game with easy to medium level of difficulty.
Note that this is not a virtual sudoku game environment, but a simple tool to read sudoku files, to check a puzzle's solvability and to provide specific hints on user request.

### Sudoku game rules
All solving features in this sudoku toolbox are based on the standard rules of the game, 
i.e. numbers from 1 to 9 need to be filled in the 9×9 matrix so that in each row, 
each column and in each 3×3 subfield, each numeral appears exactly one time.
